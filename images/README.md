# Images

Replace each file below with your real hotel photos. **Keep the same filename** — no code changes needed.

## Required files

| File | Used for |
|------|----------|
| `hero.jpg` | Homepage background (1920×1080 recommended) |
| `og-cover.jpg` | Social sharing preview (1200×630) |
| `room-standard.jpg` | Standard Single Room card |
| `room-deluxe.jpg` | Deluxe Single Room card |
| `room-luxury.jpg` | Luxury Single Room card |
| `room-studio-suite.jpg` | Deluxe Studio Suite card |
| `room-suite.jpg` | Suite, City View card |
| `restaurant-1.jpg` | Restaurant section (main) |
| `restaurant-2.jpg` | Restaurant section (secondary) |
| `conference-1.jpg` | Conference Hall section (main) |
| `conference-2.jpg` | Conference Hall section (secondary) |
| `gallery-01.jpg` … `gallery-20.jpg` | Gallery grid (800×800 or larger) |

## Tips

- Format: JPG or WebP
- Compress before upload (aim for &lt; 500 KB per image)
- Use landscape orientation for hero, rooms, restaurant, conference
- `og-cover.jpg` can be a crop of `hero.jpg`
